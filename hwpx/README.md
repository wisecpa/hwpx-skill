# HWPX 스킬 플러그인

한글(HWP/HWPX) 문서를 Claude Code 및 Claude Cowork에서 직접 생성·편집·자동화하는 스킬입니다.

## 사전 설치 필수

이 플러그인을 사용하기 전에 **pyhwpxlib** 패키지를 반드시 설치해야 합니다.

```bash
pip install pyhwpxlib
```

## 기능

| 기능 | 설명 |
|------|------|
| 새 문서 만들기 | 테마 선택 후 보고서·제안서·공문 생성 |
| 기존 문서 편집 | HWP/HWPX 파일 내용 수정 (양식 유지) |
| 양식 자동화 | 등록된 양식에 데이터 채우기 |
| 공문(기안문) 작성 | 행정안전부 편람 기준 자동 준수 |
| HWP → HWPX 변환 | 구형 HWP 파일을 HWPX로 변환 |
| 문서 분석 | 텍스트·표·이미지 추출 |

## 사용법

Claude Code 또는 Cowork에서 hwp/hwpx 파일을 언급하거나 아래 키워드를 사용하면 자동으로 활성화됩니다.

- `hwp`, `hwpx`, `한글 파일`, `한글 문서`, `한컴`
- `공문`, `기안문`, `양식 채우기`
- `/hwpx` 명령어로 직접 호출

## 버전

- pyhwpxlib 0.18.3 기준
- 플러그인 버전: 0.1.0
